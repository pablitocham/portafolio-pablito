Sobre Mí

Soy desarrollador web enfocado principalmente en Frontend, especializado en crear interfaces modernas, claras y funcionales utilizando tecnologías como HTML, CSS, JavaScript y React. También trabajo con Node.js y MongoDB para construir aplicaciones completas cuando el proyecto lo requiera.

Desarrollé una aplicación web para la gestión y cálculo de licencias utilizada en el área de Recursos Humanos de la Facultad de Arquitectura, Planeamiento y Diseño UNR sector en el que actualmente trabajo. Este proyecto surgió a partir de una necesidad real del área y me permitió implementar una solución que optimiza procesos administrativos.

Mi experiencia como responsable del área de Recursos Humanos me permitió comprender en profundidad los procesos internos de las organizaciones. Hoy combino ese conocimiento con el desarrollo web para crear herramientas digitales orientadas a resolver problemas concretos.

Me encuentro en constante aprendizaje, y busco oportunidades donde pueda seguir creciendo como desarrollador que permitan crecer tanto personal como profesionalmente.
